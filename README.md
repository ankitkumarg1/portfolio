# Ankit Kumar Gupta — Portfolio

Personal portfolio website for **Ankit Kumar Gupta**, Senior iOS Developer with 7+ years of experience.

🌐 **Live Site:** [https://ankitkumarg1.github.io](https://ankitkumarg1.github.io)

---

## Tech Stack

- Pure HTML, CSS, JavaScript — no frameworks, no dependencies
- Google Fonts (Syne + Space Mono)
- Fully responsive (mobile-first)
- Animated with CSS transitions + IntersectionObserver

## Structure

```
/
├── index.html          # Main page
├── css/
│   └── style.css       # All styles
├── js/
│   └── main.js         # Scroll effects, animations, mobile menu
└── README.md
```

## Setup

1. Clone the repository
2. Open `index.html` in a browser to preview locally
3. Push to GitHub — GitHub Pages serves it automatically from the root

## Customization

- Update App Store links in `index.html` (search for `href="#"` in project cards)
- Replace contact info if needed
- Add a profile photo by placing `assets/profile.jpg` and referencing it in the hero section
